**2. 三大区域详解（10分钟）**

**区域1：功能区**

| 按钮 | 功能 |
|------|------|
| 导入 | 导入已导出的Agent配置文件 |
| 导出 | 导出当前Agent配置，便于备份或分享 |
| 试运行 | 测试Agent运行效果 |
| 分享 | 分享Agent给其他用户 |

| 保存 | 保存当前配置 |
| 发布 | 发布Agent为API服务 |

**区域2：模块区**

**系统模块**（最常用）：
```
用户提问      - 获取用户输入，所有Agent的入口
智能对话      - 调用大模型生成回复
信息分类      - 将用户问题分类
信息提取      - 从输入中提取结构化信息
信息加工      - 对信息进行格式转换
确定回复      - 输出固定的回复内容
知识库搜索    - 在知识库中检索相关内容
```

**其他模块**：
```
智能体 - 调用已上线的其他Agent（子智能体）
工具   - 官方工具 + 自定义工具（HTTP调用等）
MCP    - 连接外部服务（SSE方式）
```

**区域3：画布区**
```
操作方式：
├── 从模块区拖拽组件到画布
├── 点击组件进行配置

├── 右键删除组件或连线
└── 鼠标滚轮缩放画布
```

#### Part 3：节点类型与连接原则（20分钟）

**1. 节点颜色与类型（8分钟）**

**四种节点类型**：

| 颜色 | 类型 | 说明 | 常见节点 |
|------|------|------|----------|
| 🟡 黄色 | 布尔型 | 只有True/False两种值 | 模块运行结束、搜索到/未搜索到、分类标签 |
| 🔵 蓝色 | 字符串 | 文本、图片、文档信息 | 文本信息、回复内容、图片信息 |
| 🟣 紫色 | 知识库结果 | 知识库检索结果 | 知识库搜索结果 |
| 🔴 红色 | 任意类型 | 可以是任何类型 | 循环模块中的元素值 |

---

**2. 连接原则（7分钟）**

**核心原则**：
```
同类型节点互相连接，不同类型不可连接

例外：
├── 红色（任意类型）可以连接任何类型
└── 某些模块有特殊转换功能
```

**正确连接示例**：
```
用户提问                    智能对话
┌──────────┐               ┌──────────┐
│ ○ 文本信息│───蓝色──────→│ ○ 信息输入│  ✓ 正确
│ (蓝色)   │               │ (蓝色)   │
└──────────┘               └──────────┘

知识库搜索                  智能对话
┌──────────┐               ┌──────────┐
│ ○ 搜索结果│───紫色──────→│ ○ 知识库  │  ✓ 正确
│ (紫色)   │               │ (紫色)   │
└──────────┘               └──────────┘
```

**错误连接示例**：
```
用户提问                    智能对话
┌──────────┐               ┌──────────┐
│ ○ 运行结束│───黄色──────→│ ○ 信息输入│  ✗ 错误
│ (黄色)   │               │ (蓝色)   │
└──────────┘               └──────────┘

正确方式：
用户提问                    智能对话
┌──────────┐               ┌──────────┐
│ ○ 运行结束│───黄色──────→│ ○ 联动激活│  ✓ 正确
│ (黄色)   │               │ (黄色)   │
└──────────┘               └──────────┘
```

### 课程内容

#### Part 1：核心模块详解（30分钟）

**模块1：用户提问模块（5分钟）**

**功能**：获取用户提问/选择内容，传输给其他模块

**界面展示**：
```
┌─────────────────────────────┐
│ 用户提问                     │
├─────────────────────────────┤
│ [模块名称] 用户提问          │
│                             │
│ □ 输入文本（默认打开）       │
│ □ 上传文档（最多9个，50MB）  │
│ □ 上传图片（最多9个，30MB）  │
│ □ 文档审查                  │
│ □ 添加按钮                  │
├─────────────────────────────┤
│ 输出：                       │
│ ○ 文本信息（蓝色）          │
│ ○ 文档信息（蓝色）          │
│ ○ 图片信息（蓝色）          │
│ ○ 模块运行结束（黄色）      │
└─────────────────────────────┘
```

**核心参数**：
| 参数 | 说明 |
|------|------|

**使用场景**：
- 所有Agent的入口模块

- 需要按钮交互的场景

```

│ ┌─────────────────────────┐ │
│ │ 你是一个智能助手...      │ │
│ └─────────────────────────┘ │
│                             │
│ 聊天上下文：[0-6]           │
│ 回复创意性：[0-1]           │
│ 回复字数上限：[100-8192]    │
│ 回复对用户可见：□           │
├─────────────────────────────┤

│ ○ 回复结束（黄色）          │
```

**核心参数详解**：

| 参数 | 范围 | 建议 | 说明 |
|------|------|------|------|

| 回复字数上限 | 100-8192 | 根据场景 | 限制回复长度 |
| 回复对用户可见 | 是/否 | 中间流程可设为否 | 是否显示给用户 |

**系统提示词模板**：
```

1. [要求1]
2. [要求2]
3. [要求3]

【注意事项】
- [注意点1]
- [注意点2]
```

**案例**：
```
场景：电力知识问答助手

你是国网四川电力的智能客服助手，专门回答电力业务相关问题。
```

```
├─────────────────────────────────────┤
```

| 重排序 | 是/否 | 精确度要求高时开启 | 会降低速度 |
