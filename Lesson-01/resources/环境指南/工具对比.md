## 八、培训场景推荐

### 8.1 场景1：网络环境良好

**推荐**：⭐⭐⭐⭐⭐ OpenCode

**理由**：
- 功能更全面
- 可使用最佳模型
- 开源免费
- 社区支持强

**配置建议**：
- 模型：Claude 3.5 Sonnet（主）+ Qwen-Plus（备）
- 登录：GitHub账号（免费Copilot）或API Key

### 8.2 场景2：网络环境受限

**推荐**：⭐⭐⭐⭐⭐ Trae-cn

**理由**：
- 国内访问稳定
- 无网络问题
- 中文支持好
- 上手简单

**配置建议**：
- 使用字节账号登录
- 选择中文场景优化

### 8.3 场景3：不确定网络环境

**推荐**：⭐⭐⭐⭐⭐ 双轨并行

**策略**：
- 主推OpenCode（功能优）
- Trae-cn作为备选（稳定优）
- 让学员根据实际情况选择

## 九、培训建议

### 9.1 Pre-Lesson准备

**必须准备**：
1. OpenCode完整安装指南（主要）
2. Trae-cn安装指南（备选）
3. 两种工具的对比说明
4. 常见问题FAQ（针对两种工具）

**建议**：
- Pre-Lesson演示OpenCode
- 提供Trae-cn作为备选方案
- 让学员提前测试选择

### 9.2 Day 4准备

**必须准备**：
1. 基于OpenCode的React开发示例
2. 基于Trae-cn的React开发示例（如需要）
3. API集成示例（通用）

**建议**：
- 大部分示例基于OpenCode
- 关键步骤提供Trae-cn对应操作

### 9.3 学员指导

**选择建议**：

```
如果网络访问国际网站通畅 → 选择OpenCode

如果对代码质量要求高 → 选择OpenCode（用Claude 3.5）
如果希望简单上手 → 选择Trae-cn
```
