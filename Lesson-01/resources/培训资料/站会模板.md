## 站会最佳实践

### 主持人职责
```
会前：
- 准备好站会模板
- 提醒成员准时参加
- 收集昨日任务完成情况

会中：
- 控制时间，每人不超过3分钟
- 引导讨论，聚焦重点
- 记录问题和行动项

会后：
- 发布站会记录
- 跟踪行动项执行
- 准备下次站会
```

### 参会者职责
```
会前：
- 准备好昨日完成情况
- 思考今日工作计划
- 整理遇到的问题

会中：
- 简洁汇报，重点突出
- 积极参与问题讨论
- 认领行动项

会后：
- 执行行动项
- 及时同步进度
- 遇到问题及时沟通
```

### 站会规则
```
时间规则：
- 每日固定时间，准时开始
- 时长15分钟，不超时
- 缺席需提前请假

发言规则：
- 每人发言不超过3分钟
- 聚焦三个问题：做了什么、计划做什么、有什么问题
- 不展开详细讨论，会后单独讨论

记录规则：
- 指定记录人
- 实时记录要点
- 会后及时发布
```
