# Lesson-01：Agentic AI 实战培训

## 课程概述

**培训时间**：2024年3月，共5天

**培训目标**：让AI种子团队成员掌握Agentic AI技术体系，能够独立开发基于Agent的智能应用

├── Day4-VibeCoding与前端开发.md       # 第4天：前端集成

## 学习路径

### Day 1：认知层 - 理论基础
- 近期AI技术趋势
- Agentic AI核心概念
- ReAct模式深入理解
- MCP协议与Skills体系
- 项目选题与分组

### Day 2-3：构建层 - Agent核心开发
- 灵知平台操作实战
- ReAct流程可视化设计
- Skills配置与编排
- 数据对接与测试
- Agent API暴露

### Day 4：深化层 - 前端开发
- Vibe Coding工具使用
- React界面开发
- API集成与调试

### Day 5：应用层 - 项目交付
- 项目完善与文档
- 成果展示与评审
- 经验总结与分享

## 项目方向

每个小组从以下三个方向中选择一个：

### 1. 智能问数
**场景**：自然语言查询电力数据
- ReAct应用：意图理解→工具选择→数据分析→可视化展示
- 前端功能：对话界面、图表展示、查询历史

### 2. 智能写作
**场景**：公文、报告、邮件智能生成
- ReAct应用：需求分析→模板检索→内容生成→格式调整
- 前端功能：富文本编辑器、模板选择、预览导出

### 3. 工单处理
**场景**：工单智能分类与处理建议
- ReAct应用：内容解析→分类推理→知识检索→建议生成
- 前端功能：工单列表、详情展示、AI分析面板

## 技术栈

- **Agent开发平台**：灵知平台
- **前端框架**：React
- **开发工具**：Trae、opencode（Vibe Coding）
- **核心技术**：Agentic AI、ReAct、MCP、Skills
- **前端组件**：ECharts（图表）、富文本编辑器

## 分组协作

**人员规模**：约70人，分为10-12组

**组内角色**：
- 产品经理（1人）：需求、设计、进度、汇报
- Agent开发者（2-3人）：灵知平台配置、ReAct设计
- 前端开发者（2-3人）：React开发、API集成

## 培训成果

每个小组完成一个端到端的智能应用：
- 可运行的Agent（通过API提供服务）
- 可交互的前端界面
- 完整的项目文档
- 5分钟演示视频
