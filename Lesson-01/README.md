# Lesson-01：Agentic AI 实战培训

人工智能种子团队培训课程（2024年3月）

---

## 📚 课程概述

**培训时间**：5天 + 3次预授课直播

**培训目标**：
- 掌握Agentic AI核心概念与ReAct模式
- 使用灵知平台构建Agent应用
- 使用React开发前端界面
- 完成智能应用端到端开发

**培训对象**：AI种子团队成员（80人）

**项目方向**：
- 智能问数：自然语言查询电力数据
- 智能写作：公文报告智能生成
- 工单处理：工单智能分类与处理建议

## 🛠 技术栈

### Agent开发

- **平台**：灵知平台（低代码拖拉拽）
- **模型**：Qwen3.5 / Qwen3
- **核心模块**：用户提问、智能对话、信息分类、信息提取、知识库搜索

### 前端开发

- **框架**：React 18+
- **工具**：OpenCode / Trae-cn（Vibe Coding）
- **构建**：Vite
- **组件库**：Ant Design / Material-UI（可选）

### 核心技术

- **ReAct模式**：推理+行动循环
- **编排方式**：规划-循环+判断+执行
- **MCP协议**：模型上下文协议
- **Skills**：技能封装

| 前端开发者 | 2-3人 | React开发、API集成 |

### 学习模式

- **全员学习**：所有内容全员学习
- **执行侧重**：执行时按角色分工
- **协作完成**：共同完成端到端项目
