下午课程：API 集成与调试

课时 5-6：API 集成（80 分钟）

1. API 服务封装

// src/services/api.js const API_CONFIG = { baseUrl: 'https://api.xxx.com/agent', uuid: 'your-uuid', authKey: 'your-auth-key' }; export const callAgent = async (question) => { const response = await fetch(`${API_CONFIG.baseUrl}/${API_CONFIG.uuid}`, { method: 'POST', headers: { 'Content-Type': 'application/json', 'Authorization': `Bearer ${API_CONFIG.authKey}` }, body: JSON.stringify({ query: question }) }); const result = await response.json(); return result.data.answer; };

2. 完整对话流程

const handleSend = async () => { // 添加用户消息 const userMessage = { id: Date.now(), type: 'user', content: input }; setMessages(prev => [...prev, userMessage]); setLoading(true); try { const answer = await callAgent(input); const botMessage = { id: Date.now() + 1, type: 'bot', content: answer }; setMessages(prev => [...prev, botMessage]); } catch (error) { const errorMessage = { id: Date.now() + 1, type: 'bot', content: '服务暂时不可用' }; setMessages(prev => [...prev, errorMessage]); } finally { setLoading(false); } };

3. 可选功能扩展

  • 清除对话：setMessages([初始消息])
  • 复制回复：navigator.clipboard.writeText(content)
  • Markdown 渲染：react-markdown 库
