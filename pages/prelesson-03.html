📹 第3次直播：Vibe Coding环境准备

时间：Day 1 前1天
总时长：90分钟（授课60分钟 + 交流30分钟）

Part 1：环境准备概述（10分钟）

必装工具

工具版本要求用途
Node.jsLTS版本（18+）JavaScript运行环境
opencode最新版AI编程助手
VS Code最新版代码编辑器（推荐）
Git最新版版本控制（可选）

什么是Vibe Coding？

传统编程：程序员手动编写每一行代码
Vibe Coding：用自然语言描述需求，AI生成代码，程序员审核调整

Part 2：Node.js安装与验证（15分钟）

下载安装

  1. 访问 https://nodejs.org
  2. 下载 LTS 版本安装包
  3. 运行安装，确认勾选 "Add to PATH"
  4. 完成安装

环境验证

node -v # 预期输出：v18.x.x 或更高 npm -v # 预期输出：9.x.x 或更高

配置npm镜像

npm config set registry https://registry.npmmirror.com

Part 3：opencode安装与配置（20分钟）

安装方式

方式1：官网下载

访问 opencode 官网，下载对应系统版本

方式2：命令行安装

# macOS/Linux curl -fsSL https://opencode.ai/install.sh | sh # Windows (PowerShell) irm https://opencode.ai/install.ps1 | iex

快速体验

第一次对话：
用户：帮我创建一个简单的HTML页面，标题是"Hello World"

opencode 会自动创建文件并展示代码

Part 4：React项目创建（15分钟）

创建项目

# 创建项目 npm create vite@latest my-first-app -- --template react # 进入项目目录 cd my-first-app # 安装依赖 npm install # 启动开发服务器 npm run dev

访问页面

打开浏览器，访问 http://localhost:5173

看到 Vite + React 页面表示成功

验收标准

  • ✅ Node.js安装成功（node -v 显示版本号）
  • ✅ opencode安装并配置
  • ✅ 能够创建React项目
  • ✅ 能够运行React项目
  • ✅ 浏览器能访问 localhost:5173
