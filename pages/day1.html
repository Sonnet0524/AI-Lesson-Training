课时 2-3：Agentic AI 概述与演进（80 分钟）

1. AI 发展的三个阶段

  • Rule-Based AI（基于规则）- if-else 逻辑，无法处理复杂场景
  • Machine Learning AI（机器学习）- 从数据中学习，需要大量标注数据
  • Agentic AI（智能体 AI）⭐ - 自主决策、工具使用、持续学习

2. Agent 四大核心特征

  • 自主性（Autonomy） - 独立决策，无需人工逐步指令
  • 规划能力（Planning） - 任务分解、步骤规划、动态调整
  • 工具使用（Tool Use） - 调用 API、数据库、搜索引擎等
  • 记忆能力（Memory） - 短期记忆（上下文）、长期记忆（历史对话）

3. ReAct 模式详解

ReAct 循环：Thought → Action → Observation → Thought → ... 案例：智能问数 Thought 1: "用户需要查询成都高新区最近 7 天的用电负荷数据" Action 1: 调用 query_power_data(region="成都高新区", date_range="7days") Observation 1: 获取数据 [{date, load}, ...] Thought 2: "分析数据趋势，发现 2/28 出现异常峰值 920MW" Action 2: 调用 query_context_info(date="2028-02-28") Observation 2: 获取上下文（招聘会 5 万 + 人流） Thought 3: "结合上下文分析，负荷峰值由大型活动导致" Final Answer: 输出分析报告

课时 4：深入 ReAct 模式（40 分钟）

ReAct vs 传统方法对比

维度 传统 Prompt ReAct
思考过程 黑盒 白盒/透明
工具使用 无法使用 主动调用
复杂任务 容易出错 分步处理
可解释性 差 强
调试难度 高 低
