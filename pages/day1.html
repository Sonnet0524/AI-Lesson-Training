课时 2-3：Agentic AI 概述（80 分钟）

Agent 四大核心特征

  • 自主性（Autonomy） - 独立决策，无需人工逐步指令
  • 规划能力（Planning） - 任务分解，步骤规划，动态调整
  • 工具使用（Tool Use） - 调用 API、数据库、搜索引擎等
  • 记忆能力（Memory） - 上下文管理，历史记录，知识积累

【第 1 轮循环】 Thought: 用户想了解成都今天的天气和穿衣建议。 我需要先查询成都今天的天气信息。 Action: { "action": "get_weather", "parameters": {"city": "成都", "date": "today"} } Observation: { "city": "成都", "weather": "多云", "temperature": "15-22°C" } 【第 2 轮循环】 Thought: 已经获取到天气信息：多云，15-22°C。 这个温度范围比较舒适，建议穿可脱的外套。 Action: 生成穿衣建议 【最终回答】 "成都今天多云，气温 15-22°C，微风。 天气比较舒适，建议穿长袖 T 恤搭配薄外套。"
