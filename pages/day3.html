上午课程：高级功能

课时 1-2：循环与代码块（80 分钟）

循环模块

功能：用于循环流程，处理数组类型数据
使用场景：批量处理多个文档、遍历查询结果、循环调用 API

功能：通过 JavaScript 或 Python 代码进行数据处理
使用场景：复杂数据转换、数学计算、字符串处理、JSON 解析/生成

课时 5-6：Agent 优化（80 分钟）

优化方向

类型优化方法
回复质量提示词优化、知识库优化、上下文优化
响应速度减少不必要模块、优化知识库检索、调整模型
用户体验对话引导、错误处理、回复格式化

课时 7-8：API 发布与测试（80 分钟）

发布流程

  1. 点击右上角"···" → "发布"
  2. 进入发布页面
  3. 选择"API 服务"标签
  4. 点击"创建链接"

API 调用示例

// React 前端调用灵知平台 Agent API const callAgent = async (question) => { const response = await fetch( 'https://api.xxx.com/agent/{uuid}', { method: 'POST', headers: { 'Content-Type': 'application/json', 'Authorization': 'Bearer {AuthKey}' }, body: JSON.stringify({ query: question, context: {} }) }); const result = await response.json(); return result.data.answer; };
