📅 Day 2：灵知平台 Agent 开发实战

课时：8 课时（上午 4 课时 + 下午 4 课时 + 晚自习 2 课时）
目标：
  • 掌握灵知平台规划页面的核心功能
  • 理解各基础模块的功能与使用场景
  • 完成各组 Agent 的初步规划与搭建
  • 实现知识库与 Agent 的对接

上午课程：Agent 设计原理与模块详解

课时 1：规划页面与模块架构（40 分钟）

1. 规划页面三大区域

  • 模块区（左侧） - 系统模块、智能体、工具、MCP 四类组件
  • 画布区（中央） - 拖拽组件、连线编排的设计区域
  • 功能区（上方） - 导入/导出、试运行、分享、发布

2. 节点类型与连接原则

● 黄色 - 布尔型 ● 蓝色 - 字符串 ● 紫色 - 知识库结果 ● 红色 - 任意类型

连接原则：同类型节点互相连接，不同类型节点不可连接

3. 模块结构

  • 左上角 - 模块输入区：连接节点将信息输入
  • 中间 - 模块设置区：设置参数、提示词等
  • 右下角 - 模块输出区：将模块执行后的信息输出

课时 2-3：核心模块详解（80 分钟）

1. 用户提问模块（入口）

  • 获取用户提问/选择内容
  • 支持文本输入、上传文档（9 个，50MB/个）、上传图片（9 个，30M/个）
  • 可添加按钮作为明确执行节点

2. 智能对话模块

  • 通过大语言模型处理并回复
  • 参数：聊天上下文（0-6 条）、回复创意性（0-1）、回复字数上限（100-8192）
  • 可连接知识库搜索结果

3. 信息分类模块

  • 将用户问题分类（如"咨询""投诉""建议""其他"）
  • 输出：标签节点（布尔型）、分类结果（JSON）
  • 场景：工单分类、问题类型识别

4. 信息提取模块

  • 从输入信息中提取目标字段
  • 配置字段 key、描述、是否必填
  • 输出：全部提取结果（JSON 字符串）

5. 知识库搜索模块

  • 在知识库中搜索相关问题与解答
  • 参数：相似度阈值（0.6-0.9）、召回数（5-10）、重排序
  • 输出：搜索到/未搜索到相关知识（布尔型）

课时 4：模块组合与流程设计（40 分钟）

常见编排模式

模式 1：简单问答 用户提问 → 智能对话 → 回复用户 模式 2：知识库问答 用户提问 → 知识库搜索 → ├─ 搜索到 → 智能对话（结合知识库） → 回复 └─ 未搜索到 → 确定回复（默认回答） 模式 3：分类处理 用户提问 → 信息分类 → ├─ 类型 A → 处理流程 A → 回复 ├─ 类型 B → 处理流程 B → 回复 └─ 其他 → 引导重新描述 模式 4：提取+API 调用 用户提问 → 信息提取 → HTTP 调用 → 信息加工 → 回复
