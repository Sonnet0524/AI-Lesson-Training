📋 课程概览

🎯 培训对象

国网四川电力人工智能种子团队成员（约 80 人）

⏱️ 培训时间

5 天集中培训
每课时 40 分钟

👥 分组方式

10-12 组
每组 6-7 人

📊 培训比例

30% 理论 + 70% 实操

🎓 培训目标

  • 掌握 Agentic AI 核心概念与 ReAct 模式
  • 使用 灵知平台构建 Agent 应用
  • 使用 React 开发前端界面
  • 完成 智能应用端到端开发

🛠️ 技术栈

Agentic AI ReAct 模式 灵知平台 React MCP Vibe Coding

📁 项目方向

智能问数

自然语言查询电力数据

  • 数据查询
  • 趋势分析
  • 可视化展示

智能写作

公文报告智能生成

  • 模板选择
  • 内容生成
  • 格式排版

工单处理

工单智能分类与处理

  • 自动分类
  • 智能分派
  • 处理建议
