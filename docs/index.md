## 课程概览

### 培训目标

- **掌握** Agentic AI 核心概念与 ReAct 模式
- **使用** 灵知平台构建 Agent 应用
- **使用** React 开发前端界面
- **完成** 智能应用端到端开发

### 培训对象

国网四川电力人工智能种子团队成员（约 80 人）

### 培训方式

- **分组协作**：12-14 组，每组 6-7 人
- **学习模式**：30% 理论 + 70% 实操
- **培训周期**：5 天集中培训 + 3 次预授课直播

| 日期 | 主题 | 主要内容 |

培训课程围绕三个真实业务场景展开：
