## 二、灵知平台模块速查

### 系统模块

| 模块 | 功能 | 典型场景 |
|------|------|----------|
| 用户提问 | 获取用户输入 | 所有Agent入口 |
| 智能对话 | 大模型生成回复 | 问答、创作 |
| 信息分类 | 问题分类 | 工单分类、流程分支 |
| 信息提取 | 提取结构化信息 | 参数提取、API调用前处理 |
| 知识库搜索 | 检索知识库 | 知识问答、文档检索 |
| 信息加工 | 格式转换 | 数据处理 |
| 确定回复 | 固定回复 | 引导、提示 |
| 循环 | 批量处理 | 多文档处理 |
| 代码块 | 自定义代码 | 复杂计算、数据转换 |

### 节点类型

| 颜色 | 类型 | 说明 | 连接规则 |
|------|------|------|----------|
| 🟡 黄色 | 布尔型 | True/False | 只能连布尔型 |
| 🔵 蓝色 | 字符串 | 文本/图片/文档 | 只能连字符串 |
| 🟣 紫色 | 知识库结果 | 检索结果 | 只能连知识库输入 |
| 🔴 红色 | 任意类型 | 任意值 | 可连任何类型 |

## 五、API调用方式

### 发布API

1. 在灵知平台设计好Agent流程
2. 点击右上角"发布"
3. 选择"API服务"
4. 创建链接获取凭证

### API凭证

| 信息 | 说明 |
|------|------|
| Uuid | API唯一标识符 |
| AuthKey | 验证密钥 |
| AuthSecret | 安全密钥 |

### 调用示例

```javascript
// React前端调用灵知平台Agent API
const callAgent = async (question) => {
  const response = await fetch('https://api.xxx.com/agent/{uuid}', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': 'Bearer {AuthKey}'
    },
    body: JSON.stringify({
      query: question,
      context: {}
    })
  });
  
  const result = await response.json();
  return result.data.answer;
};
```

## 六、Vibe Coding工具

### opencode使用技巧

**描述需求**：
```
✅ 具体：描述功能、样式、交互
✅ 渐进：从简单开始，逐步迭代
✅ 明确：说明不想要的和想要的
✅ 上下文：提供必要的背景信息
```

| 任务 | 提示词模板 |

| 创建组件 | 创建一个[组件名]组件，功能是[描述]，props包括[列表] |

## 七、项目选题技术架构

```

Agent：用户提问 → 信息分类 → 知识库搜索 → 智能对话
核心模块：信息分类、知识库搜索、智能对话

```
前端：React + 列表组件
