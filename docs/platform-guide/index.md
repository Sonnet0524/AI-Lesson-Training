# 灵知平台使用指南

## 平台概述

灵知平台是一个低代码 Agent 开发平台，支持快速构建智能应用。

---

## 快速开始

### 创建 Agent

1. 登录灵知平台
2. 点击"创建智能体"
3. 配置基本信息：
   - 名称：清晰描述用途
   - 描述：详细功能说明
   - 开场白：用户首次看到的消息

### 基础配置

| 配置项 | 说明 | 建议 |
|--------|------|------|
| 智能体头像 | 可自定义 | 使用相关图标 |
| 智能体名称 | 命名清晰 | 包含用途关键词 |
| 智能体描述 | 功能说明 | 50-100字 |
| 开场白 | 引导用户 | 简洁友好 |
| 问题建议 | 预设问题 | 最多6个 |

---

## 核心模块

### 1. 用户提问

**功能**：接收用户输入

**配置**：
- 输入提示语：引导用户提问
- 默认值：可选

**节点输出**：字符串类型（用户消息）

---

### 2. 智能对话

**功能**：LLM 对话交互

**参数**：
| 参数 | 说明 | 推荐值 |
|------|------|--------|
| 模型选择 | LLM模型 | Qwen-Plus |
| 系统提示词 | 角色设定 | 详细描述Agent功能 |
| 温度 | 创意性 | 0.7 |
| 最大长度 | 输出长度 | 2000 |

**最佳实践**：
- 系统提示词要详细具体
- 包含角色、任务、约束
- 提供示例对话

---

### 3. 知识库搜索

**功能**：RAG 检索增强

**参数**：
| 参数 | 说明 | 推荐值 |
|------|------|--------|
| 知识库 | 文档集合 | 根据场景选择 |
| 相似度阈值 | 召回精度 | 0.6 |
| 召回数 | 结果数量 | 5 |
| 重排序 | 精排优化 | 可选 |

**配置步骤**：
1. 创建知识库
2. 上传文档
3. 等待索引完成
4. 模块中选择知识库

---

### 4. 信息分类

**功能**：文本分类

**配置**：
- 分类标签：定义类别
- 分类规则：可选

**输出**：分类结果（字符串）

---

### 5. 信息提取

**功能**：结构化提取

**配置**：
- 提取字段：定义要提取的信息
- 提取规则：提取逻辑

**输出**：JSON 格式

## 最佳实践

### 提示词设计

**模板**：
```

你是一个[角色描述]

# 任务
[具体任务说明]

# 约束
- [约束1]
- [约束2]

# 示例
用户：[示例输入]
助手：[示例输出]
```

---

### 性能优化

**建议**：
- 合理设置召回数
- 避免过度重排序
- 控制上下文长度
- 使用缓存机制

---

## 常见问题

### Q1: Agent 回复不准确？

**排查**：
1. 检查系统提示词
2. 优化知识库内容
3. 调整参数配置
4. 增加示例对话

---

### Q2: 响应速度慢？

**优化**：
1. 减少召回数
2. 关闭重排序
3. 简化流程
4. 使用缓存

---

### Q3: 如何调试 Agent？

**方法**：
1. 使用测试对话框
2. 查看中间结果
3. 单步调试
4. 日志分析
