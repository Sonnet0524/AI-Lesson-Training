# Pre-Lesson 第1次直播：灵知平台入门

## 基本信息

**时间**：Day 1 前3天

**总时长**：90分钟（授课60分钟 + 交流30分钟）

**学习目标**：
- 完成灵知平台账号注册和登录

### Part 1：平台介绍与账号准备（15分钟）

#### 1. 灵知平台定位（5分钟）

**平台简介**：
```
灵知平台是什么？
├── 企业级AI应用开发平台
├── 低代码/无代码方式构建Agent
├── 支持知识库、工具调用、API发布
└── 国网四川电力AI应用基础设施
```

**核心能力**：
| 能力 | 说明 |
|------|------|
| Agent编排 | 可视化拖拽设计智能体流程 |

Q：企业邮箱无法注册？

Q：忘记密码怎么办？

#### 2. 三大区域详解（10分钟）

**区域1：功能区**

| 按钮 | 功能 |
|------|------|
| 导入 | 导入已导出的Agent配置文件 |
| 导出 | 导出当前Agent配置，便于备份或分享 |
| 试运行 | 测试Agent运行效果 |
| 分享 | 分享Agent给其他用户 |
| 删除 | 删除当前Agent |
| 保存 | 保存当前配置 |
| 发布 | 发布Agent为API服务 |

**区域2：模块区**

**系统模块**（最常用）：
```
用户提问      - 获取用户输入，所有Agent的入口
智能对话      - 调用大模型生成回复
信息分类      - 将用户问题分类
信息提取      - 从输入中提取结构化信息
信息加工      - 对信息进行格式转换
确定回复      - 输出固定的回复内容
知识库搜索    - 在知识库中检索相关内容
```

**其他模块**：
```
智能体 - 调用已上线的其他Agent（子智能体）
工具   - 官方工具 + 自定义工具（HTTP调用等）
MCP    - 连接外部服务（SSE方式）
```

**区域3：画布区**
```
操作方式：
├── 从模块区拖拽组件到画布
├── 点击组件进行配置
├── 拖拽连线连接组件
├── 右键删除组件或连线
└── 鼠标滚轮缩放画布
```

---

#### 3. 模块结构（5分钟）

每个模块由三部分组成：
```
┌─────────────────────────┐
│  模块输入区（左上角）    │  
│  ○ 联动激活             │  ← 上一模块完成后触发
│  ○ 任一激活             │  ← 任一输入触发
│  ○ 信息输入             │  ← 接收上一模块的输出
├─────────────────────────┤
│  模块设置区（中间）      │  ← 配置参数、提示词
│  [模型选择]             │
│  [系统提示词]           │
│  [其他参数...]          │
├─────────────────────────┤
│  模块输出区（右下角）    │  ← 执行后的输出
│  ○ 回复结束             │  ← 是否已回复完成
│  ○ 回复内容             │  ← 生成的回复文本
│  ○ 模块运行结束         │  ← 模块执行完毕
└─────────────────────────┘
```

#### 4. 第一个Agent演示（7分钟）

**演示目标**：创建一个简单的问答Agent

**步骤**：
```
步骤1：点击"新建Agent"
步骤2：拖入"用户提问"模块到画布
步骤3：拖入"智能对话"模块到画布
步骤4：连接两个模块
       - 从"用户提问"的"模块运行结束"拖线到"智能对话"的"联动激活"
       - 从"用户提问"的"文本信息"拖线到"智能对话"的"信息输入"
步骤5：配置"智能对话"模块
       - 系统提示词："你是一个友好的助手"
步骤6：点击"试运行"
步骤7：在对话框输入"你好"，查看回复
步骤8：点击"保存"
```

**讲师演示**：
- 完整演示以上步骤
- 强调连线要点
- 演示试运行和保存

### Part 3：节点类型与连接原则（20分钟）

#### 1. 节点颜色与类型（8分钟）

**四种节点类型**：

| 颜色 | 类型 | 说明 | 常见节点 |
|------|------|------|----------|
| 🟡 黄色 | 布尔型 | 只有True/False两种值 | 模块运行结束、搜索到/未搜索到、分类标签 |
| 🔵 蓝色 | 字符串 | 文本、图片、文档信息 | 文本信息、回复内容、图片信息 |
| 🟣 紫色 | 知识库结果 | 知识库检索结果 | 知识库搜索结果 |
| 🔴 红色 | 任意类型 | 可以是任何类型 | 循环模块中的元素值 |

---

#### 2. 连接原则（7分钟）

**核心原则**：
```
同类型节点互相连接，不同类型不可连接

例外：
├── 红色（任意类型）可以连接任何类型
└── 某些模块有特殊转换功能
```

**正确连接示例**：
```
用户提问                    智能对话
┌──────────┐               ┌──────────┐
│ ○ 文本信息│───蓝色──────→│ ○ 信息输入│  ✓ 正确
│ (蓝色)   │               │ (蓝色)   │
└──────────┘               └──────────┘

知识库搜索                  智能对话
┌──────────┐               ┌──────────┐
│ ○ 搜索结果│───紫色──────→│ ○ 知识库  │  ✓ 正确
│ (紫色)   │               │ (紫色)   │
└──────────┘               └──────────┘
```

**错误连接示例**：
```
用户提问                    智能对话
┌──────────┐               ┌──────────┐
│ ○ 运行结束│───黄色──────→│ ○ 信息输入│  ✗ 错误
│ (黄色)   │               │ (蓝色)   │
└──────────┘               └──────────┘

正确方式：
用户提问                    智能对话
┌──────────┐               ┌──────────┐
│ ○ 运行结束│───黄色──────→│ ○ 联动激活│  ✓ 正确
│ (黄色)   │               │ (黄色)   │
└──────────┘               └──────────┘
```

#### 3. 连接实操练习（5分钟）

**练习任务**：正确连接以下模块

```
模块：用户提问 → 知识库搜索 → 智能对话

需要连接：
1. 用户提问"模块运行结束" → 知识库搜索"联动激活"
2. 用户提问"文本信息" → 知识库搜索"查询内容"
3. 知识库搜索"模块运行结束" → 智能对话"联动激活"
4. 知识库搜索"搜索结果" → 智能对话"知识库搜索结果"
```

- 解答连接问题
