### 课时2-4：Agent搭建实战（120分钟）

#### 学习目标
- 完成Agent基础配置
- 搭建核心流程
- 实现基本功能

#### 实践任务

**步骤1：基础配置（15分钟）**
- 设置Agent名称、描述
- 上传头像或选择默认
- 设置开场白
- 配置问题建议（3-6个）

**步骤2：创建知识库（如需）（25分钟）**
- 上传文档（pdf/doc/docx/txt）
- 设置解析方法
- 等待解析完成
- 进行检索测试

**步骤3：搭建流程（80分钟）**
- 拖入所需模块
- 连接各模块
- 设置参数
- 试运行调试

#### 各选题指导

**选题1：智能问数**
```
流程设计：
用户提问 → 信息提取（提取查询意图）→ HTTP调用（查询数据）→ 信息加工（格式化）→ 智能对话（分析+回复）

关键点：
- 信息提取：准确识别时间、指标、维度
- HTTP调用：对接数据接口
- 可视化：调用图表生成工具
```

**选题2：智能写作**
```
流程设计：
用户提问 → 信息分类（文档类型）→ 
  ├─ 报告类 → 知识库搜索（模板）→ 信息提取（数据需求）→ HTTP调用 → 智能对话（生成）
  └─ 公文类 → 知识库搜索（模板）→ 智能对话（生成）

关键点：
- 分类准确：区分文档类型
- 模板管理：知识库存储模板
- 内容质量：提示词优化
```

**选题3：工单处理**
```
流程设计：
用户提问 → 信息分类（工单类型）→ 
  ├─ 故障报修 → 知识库搜索（处理流程）→ 智能对话（生成建议）
  ├─ 业务咨询 → 知识库搜索（FAQ）→ 智能对话（回复）
  └─ 投诉建议 → 信息提取（关键信息）→ 确定回复（引导流程）

关键点：
- 分类准确率
- 知识库完善
- 多分支流程
```

#### 讲师巡场指导

**常见问题**：
1. 模块连接错误（检查节点类型）
2. 知识库搜索不到（调整相似度阈值）
3. 提取字段不完整（优化提示词）
4. 流程不执行（检查激活条件）
5. HTTP调用失败（检查接口格式）

### B. 参数建议值

| 场景 | 回复创意性 | 相似度阈值 | 召回数 |
|------|------------|------------|--------|
| 知识库问答 | 0 | 0.7-0.8 | 5-10 |
| 创作生成 | 0.6-0.8 | - | - |
| 工单分类 | 0 | - | - |
| 信息提取 | 0 | - | - |

### C. 常用提示词模板

**信息分类**：
```
请判断用户问题属于哪种类型，可选类型：咨询、投诉、建议、其他。
只输出类型名称，不要输出其他内容。
```

**信息提取**：
```
请从用户输入中提取以下信息：
- 会议室名称
- 预约时间
- 预约人姓名

以JSON格式输出，如：{"room":"xxx","time":"xxx","person":"xxx"}
```

**知识库问答**：
```
请根据以下知识库内容回答用户问题：
{{知识库搜索结果}}

如果知识库内容与问题无关，请回答"抱歉，暂时无法回答这个问题"。
```
