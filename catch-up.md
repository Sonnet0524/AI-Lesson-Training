## 五、常见操作

### 5.1 修改课程内容

1. 找到对应的 Markdown 文件
2. 同时修改 HTML 页面（`pages/` 目录）
3. 测试链接有效性
4. Git commit + push

### 5.2 添加新项目设计

1. 创建 `projects/项目名-项目设计.md`
2. 创建 `pages/project-xxx.html`
3. 更新 `index.html` 添加链接
4. Git commit + push

### 5.3 修改 HTML 页面

1. 找到 `pages/` 下对应文件
2. 修改内容
3. 检查所有链接
4. Git commit + push

